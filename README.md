# Hotel Room Reservation System

A React + Vite implementation of the SDE-3 assessment.

## Features
- Book up to 5 rooms
- Same-floor priority allocation
- Cross-floor optimization using travel-time calculation
- Random occupancy generation
- Reset bookings
- Visual room visualization

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack
- React
- Vite
- CSS
